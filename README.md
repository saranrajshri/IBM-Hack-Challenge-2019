# Installation

# Setup Flask

Install Dependencies

```bash
pip3 install flask
pip3 install tweepy
pip3 install praw
pip3 install scikit-learn
pip3 install numpy
pip3 install pandas
pip install  "ibm-watson>=3.0.3"
```
# Run Flask

```bash
cd flask
export FLASK_APP=server.py
flask run
```

# Setup React

```
cd client
npm install
```

# Run React
```bash
npm start
```

The React Server Will Run On localhost:3000
