# Installation

# Setup Flask

Install Dependencies

```bash
cd flask
sudo pip3 install -r "requirements.txt"
pip3 install  "ibm-watson>=3.0.3"
```
# Run Flask

```bash
cd flask
export FLASK_APP=server.py
flask run
```

# Setup React

```
cd client
npm install
```

# Run React
```bash
npm start
```

The React Server Will Run On localhost:3000
